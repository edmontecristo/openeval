# OpenEval Logo Design Brief

## Concept
A scientific, modern logo combining:
- Microscope icon (symbolizing "evaluation" and "analysis")
- Terminal/console elements (CLI-first philosophy)
- Green checkmark or progress indicator (testing/success)

## Style
- Flat, minimalist design
- Color palette: Emerald green (#10b981) + Slate blue (#6366f1)
- White background for versatility
- Appropriate for GitHub README and documentation
